# Documentation Images

This directory contains images for documentation.

## Recommended Images

Add the following images to enhance documentation:

- `architecture.png` - System architecture diagram
- `pipeline.png` - DeepStream pipeline flow
- `dashboard.png` - Grafana dashboard screenshot
- `heatmap-example.png` - Heatmap visualization
- `queue-monitor.png` - Queue monitoring UI
- `jetson-setup.png` - Jetson device setup

## Image Guidelines

- Format: PNG or SVG preferred
- Resolution: 1200px width minimum
- File size: < 500KB per image
- Naming: lowercase with hyphens

## Creating Architecture Diagrams

Tools recommended:
- [draw.io](https://draw.io) - Free online diagram tool
- [Excalidraw](https://excalidraw.com) - Hand-drawn style
- [Mermaid](https://mermaid.js.org) - Code-based diagrams
